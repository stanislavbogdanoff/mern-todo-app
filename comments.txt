Много времени было потрачено на создание Update функциональности.

Проблема была с синтаксисом методов PUT и PATCH.

Выход для PUT был найден на гитхабе в похожем проекте, PATCH сам сообразил.

Function for PUT request in TodoSlice.js accepts a single argument, an object with id and text; 
Then in TodoService.js this object is destructured and passes id and text as first and second args to axios.put request.

Function for PATCH recievs id as a single argument in TodoSlice.js. 
Then in TodoService.js it passes it as the first arg but also passes an empty object as the second arg.
